## 🚀 Quick Start - ทดสอบประวัติการแก้ไข

**เวลาที่ใช้:** ~5 นาที

### 📍 Step 1: เข้า Dashboard (1 นาที)

1. เปิด Browser ไปหน้า Dashboard
2. ตรวจสอบว่าแสดงรายการคำสั่งพิมพ์ได้ปกติ

### 📍 Step 2: เลือกคำสั่ง (1 นาที)

1. ค้นหาคำสั่งที่อยากทดสอบ (ใช้ search ก็ได้)
2. หรือสร้างคำสั่งใหม่ได้

### 📍 Step 3: แก้ไขข้อมูล (1 นาที)

1. คลิกปุ่ม **✏️ (Edit)** สีน้ำเงิน
2. เปลี่ยนข้อมูล เช่น:
   - เปลี่ยน **Quantity** (จำนวน) จาก 100 เป็น 150
   - หรือเปลี่ยน **Production Date**
3. คลิกปุ่ม **บันทึก**

### 📍 Step 4: ดูประวัติ (2 นาที)

1. ค้นหาการ์ดของคำสั่งอีกครั้ง
2. **ลูกน้อย 👇** - ดูหลังจากเพิ่มจำนวน/วันที่ผลิต
3. หากมีการแก้ไข ควรเห็น:

```
📅 ประวัติการแก้ไข (2) ▼

[แก้ไข] 25/04/2567, 15:45 ⭐ ล่าสุด
โดย: ชื่อคุณ (EMP001)
แก้ไข: จำนวน: 100 ➡️ 150
───────────────────────

[สร้าง] 25/04/2567, 14:30
โดย: ชื่อคนสร้าง (EMP000)
สร้างคำสั่งพิมพ์ใหม่
```

### ✅ ทดสอบสำเร็จ!

ถ้าเห็น:
- ✓ ปุ่ม "ประวัติการแก้ไข" 
- ✓ รายการประวัติแสดงออกมา
- ✓ เห็นชื่อผู้แก้ไข
- ✓ เห็นเวลาแก้ไข
- ✓ เห็นสิ่งที่เปลี่ยนแปลง

🎉 **ทดสอบผ่าน!**

### ⚡ ทดสอบเพิ่มเติม (Optional)

**ลองทดสอบการตรวจสอบ:**
1. กด ✓ (Verify) ปุ่มเขียว
2. ยืนยัน
3. ดูประวัติ → ควรเห็น [ตรวจสอบ]

**ลองทดสอบการยกเลิก:**
1. สร้างคำสั่งใหม่
2. กด ✕ (Cancel) ปุ่มแดง
3. ระบุเหตุผล
4. ดูประวัติ → ควรเห็น [ยกเลิก]

### 🐛 ถ้าไม่เห็นประวัติ?

1. **ล้าง Cache:**
   - `Ctrl + Shift + Delete` (บน Windows)
   - `Cmd + Shift + Delete` (บน Mac)
   - เลือก "All time"

2. **F5 Refresh**

3. **ตรวจสอบ Console:**
   - `F12` → Console tab
   - ดู Error messages (สีแดง)

4. **ตรวจสอบ Network:**
   - `F12` → Network tab
   - ดูการเรียก `/api/orders/.../history`
   - ตรวจสอบ Response (ควรเป็น 200 OK)

### 💡 Tips

- **ประวัติแสดงเก่าสุดไป** (newer first)
- **ไม่เห็นประวัติ?** แสดงว่ายังไม่มีการแก้ไข
- **ขยายหรือยุบ:** คลิกปุ่มเดิมอีกครั้ง
